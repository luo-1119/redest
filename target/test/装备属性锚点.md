#角色特征属性

角色类型    特征属性    说明
攻击类      攻击        攻击型角色的主属性，装备中额外加权
防御类      防御        防御型角色的主属性
辅助类      生命        辅助/治疗型角色的主属性

#装备部位与权重

部位      权重分配
武器      1/4攻击 + 1/4特征属性 + 1/6暴击
防具      1/4防御 + 1/4生命 + 1/6格挡
饰品      1/4攻击 + 1/4生命 + 1/6暴击
功法      1/4攻击 + 1/4防御 + 1/6格挡

特征属性释义：装备的"1/4特征属性"根据角色类型决定加成为攻击/防御/生命。

#装备等级与属性公式

装备等级n仅取10的倍数（10,20,30,...,100）
装备属性 = 成长资质 * (n+5) * 1.023293^(n+5) * 波动系数 * 对应权重

成长资质：攻击=12, 生命=98, 防御=5, 暴击=10, 格挡=10
波动系数：[0.5, 1.75]，每个属性独立随机

#品质跨级对比

低品质高等级 vs 高品质低等级：
    装备LV60白武(中值980) vs 装备LV40彩武(基准190.2*3.2*1.125=685)
    → LV60白武胜出(980>685)，等级压制明显

建议：优先提升装备等级，同等级下追求品质。

#设计总结

- 装备等级以10为阶梯，公式与角色基础属性同构
- 波动[0.5,1.75]提供3.5倍个体差异，同品质同等级间也有区分度
- 品质倍率1.0~3.2提供7级梯度，彩色≈3.2倍白色，跨品质差距巨大
- 特征属性使三类角色在同部位装备上有不同收益，增强Build差异
