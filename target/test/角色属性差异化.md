#三大类型

类型      特征加成                    说明
攻击类    攻击资质*1.05               主攻，高攻击
防御类    防御资质*1.05               主守，高防御
辅助类    生命资质*1.05               主治疗，高生命

加成仅作用于成长资质项，初始属性不变。
类型差异温和（仅5%），主要差异体现在装备特征属性权重和技能机制。

#五个品质（全局资质乘数）

品质      资质总倍率      说明
白色      1.0             普通，标准成长
绿色      1.1             良好
蓝色      1.2             优秀
紫色      1.4             卓越
橙色      1.8             稀有

品质影响所有五项成长资质（攻击/生命/防御/暴击/格挡），全局乘算。
初始属性不受品质影响，低等级时品质差距较小、高等级时差距急剧放大。

#跨品质对战示例（LV60，攻击类橙 vs 辅助类白）

攻击方：攻击类橙 ATK=5537，暴击率=44%
防御方：辅助类白 HP=25559，DEF=1244

普攻伤害 = (5537 - 1244) = 4293
击杀所需普攻 = 25559 / 4293 = 6.0次

对比同品质14次，品质压制使TTK缩短57%。

#四个评级（B/A/S/L）

评级      稀有度    技能机制强度        说明
B         普通      基础技能            仅锚点倍率技能
A         稀有      强化技能            技能附带轻微附效（如小幅减防）
S         超稀有    高级技能            技能附带强力附效或多目标选择
L         传说      终极技能            复合技能或特殊机制

评级不直接影响属性数值，仅影响技能池与技能机制复杂度。
具体技能需在技能列表中按评级分级设计。

#类型定位

类型      核心资质    装备侧重        九宫格位置    技能倾向
攻击类    攻击+5%     武器+功法       中排/后排    单体爆发、AOE
防御类    防御+5%     防具+功法       前排         嘲讽/护盾/减伤
辅助类    生命+5%     防具+饰品       后排         治疗/Buff/净化

#品质成长曲线

品质差距随等级放大（因乘在成长项上）：
    LV1:  白ATK=133 vs 橙ATK=143（差8%）
    LV60: 白ATK=3130 vs 橙ATK=5537（差77%）
    LV100:白ATK=12713 vs 橙ATK=22708（差79%）

建议：低等级优先等级，高等级品质质变。
