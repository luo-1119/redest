#宝石符文系统

来源：宝石机制.md

#新增属性

属性      公式                      作用
韧性      暴击资质*n*1.023293^n      等数值抵消敌方暴击属性
破击      格挡资质*n*1.023293^n      等数值抵消敌方格挡属性
穿透      攻击资质*n*1.023293^n*1/5  造成伤害时无视等量防御，且最终伤害不低于穿透值

#符文类型（每个角色可附加4个）

力量符文      1/2攻击 + 1/2防御      攻击+防御      均衡攻防
体质符文      1/2防御 + 1/2生命      防御+生命      坦克向

穿透符文      1/5穿透                攻击资质      破防+伤害保底
暴击符文      1暴击                  暴击资质      提升暴击率
格挡符文      1格挡                  格挡资质      提升格挡率

#韧性/破击的抵消机制

有效暴击属性 = max(0, 攻击方暴击属性 - 防御方韧性)
有效格挡属性 = max(0, 防御方格挡属性 - 攻击方破击)
有效暴击率 = 有效暴击属性 / 10000
有效格挡率 = 有效格挡属性 / 10000

示例（LV60，双方基础暴击=格挡=2489=24.89%）：

场景                              攻击方有效暴击    防御方有效格挡
双方无符文                        24.89%           24.89%
防御方带韧性符文(2389)            2489-2389=100→1%  24.89%
攻击方带破击符文(2389)            24.89%           2489-2389=100→1%
双方各带对应符文                  互相抵消，回归24.89%

单符文即可将对方暴击/格挡压至几乎为零，韧性/破击是强对抗属性。

#穿透机制

实际伤害计算（含穿透）：
    基础伤害 = (bas - max(0, 承受方防御 - 穿透)) * kr * kf
    最终伤害 = max(穿透, 基础伤害)

即：穿透先削减对方有效防御，再作为最终伤害的下限保底。

示例（LV60，攻击方ATK=2987，穿透=573，防御方DEF=1244）：
    无穿透：伤害 = 2987 - 1244 = 1743
    有穿透：有效DEF = max(0, 1244-573) = 671
            伤害 = max(573, 2987-671) = max(573, 2316) = 2316

穿透使伤害从1743提升至2316（+33%），对高防目标效果尤为显著。

#4符文配置策略

配置名            符文组合                      适用角色    效果
均衡流            力量+暴击+格挡+穿透           通用        攻防双修
暴击流            暴击+力量+破击+穿透           攻击类      极限输出
坦克流            体质+格挡+韧性+力量           防御类      极致生存
穿透流            穿透+破击+暴击+力量           攻击类      针对高防
反暴流            韧性+体质+格挡+力量           辅助类      防暴击秒杀

#符文等级与角色等级的关系

符文等级n独立于角色等级，类似装备等级。
符文等级建议与装备等级同阶梯（10的倍数）。
符文属性受角色品质倍率影响（符文公式中的"对应成长资质"已含品质乘数）。

#符文系统的设计意义

- 新增韧性/破击/穿透三角对抗，丰富暴击/格挡的克制链
- 4符文槽=4个策略选择，Build自由度大幅提升
- 穿透提供"保底伤害"，避免高防目标完全免疫
- 符文数值量级与装备相当，是后期核心养成线之一
